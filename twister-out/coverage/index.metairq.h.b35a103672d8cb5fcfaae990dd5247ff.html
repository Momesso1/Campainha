53 /**
54 * @brief Recover a preempted cooperative thread after a metairq completes.
55 *
56 * After the run-queue winner (@p thread) has been chosen, check whether a
57 * cooperative thread was preempted by a metairq and should be resumed instead.
58 * Returns the adjusted thread pointer (unchanged when metairq logic is
59 * disabled or no recovery is needed).
60 *
61 * @param thread Best candidate chosen from the run queue.

86 /**
87 * @brief Return true if _current may be re-queued after being context-switched.
88 *
89 * When a metairq thread preempts a cooperative thread, the preempted thread
90 * must not be put back on the run queue until the metairq completes;
91 * otherwise a lower-priority thread might steal it.
92 *
93 * @return true if it is safe to re-queue _current, false otherwise.

95 static inline bool metairq_current_requeue_allowed(void)
